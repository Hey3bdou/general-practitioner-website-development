# موقع العيادة - طبيب عام

موقع ويب متكامل لعيادة طبيب عام يتضمن نظام حجز المواعيد وإدارة المقالات والأخبار.

## الميزات الرئيسية

### 1. نظام حجز المواعيد
- حجز المواعيد إلكترونياً
- اختيار التاريخ والوقت المناسب
- تأكيد أو إلغاء المواعيد
- تتبع حالة المواعيد

### 2. نظام المقالات
- كتابة ونشر المقالات الطبية
- تصنيف المقالات حسب الموضوع
- البحث في المقالات

### 3. نظام الأخبار
- نشر أخبار العيادة
- إعلانات العروض الخاصة

### 4. نظام المصادقة
- تسجيل المرضى الجدد
- تسجيل الدخول

### 5. نظام الاتصال
- نموذج الاتصال
- معلومات التواصل
- الأسئلة الشائعة

## التقنيات المستخدمة

- **Next.js 16** - إطار عمل React
- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - لغة برمجة
- **Tailwind CSS 4** - إطار عمل CSS
- **Drizzle ORM** - إدارة قاعدة البيانات
- **PostgreSQL** - قاعدة البيانات

## التشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل التطبيق
npm run dev
```

فتح المتصفح: `http://localhost:3000`
